this app is a Flask application which manages a machine-learning based text classifier and speak to both MongoDB and MySQL,
it include three steps
1. run the classification experiment, and the results, models and configuration will be stored into MongoDB
2. Retrieve results for the experiment, the top 3 fastest model and top 3 highest score model will be display on the browser
3. put a text, system will predict whether it is about covid-19 or not
